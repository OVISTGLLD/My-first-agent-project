# Agent Skills

摄影研究站的 Agent Skills 系统。每个 Skill 是一个独立的 Markdown 文件，定义了可复用的工作流程。

## 目录

| Skill | 说明 |
|-------|------|
| [project_analyst.md](project_analyst.md) | 摄影项目分析 — 输入项目信息，输出主题分析、母题分析、摄影师/摄影书推荐、项目发展建议 |

## Skill 规范

每个 Skill 文件必须包含以下结构：

### 1. 角色定义

明确该 Skill 扮演的角色——它是做什么的？在什么场景下使用？

### 2. 输入

列出调用该 Skill 时需要的所有输入字段，包括字段名称、类型和说明。

### 3. 输出

明确该 Skill 的输出内容。每部分输出应该标题清晰、结构固定、可被机器或人类读取。

### 4. 工作流程

描述该 Skill 的处理步骤：
- 第一步做什么
- 第二步做什么
- 每一步的依据是什么

### 5. 输出格式

指定输出的格式规范（Markdown / JSON / YAML 等），确保不同调用产生一致的结果。

## 设计原则

- **可复用** — 每个 Skill 是自包含的，可在不同项目中重复调用
- **可组合** — 一个 Skill 的输出可以作为另一个 Skill 的输入
- **具体而非泛泛** — 每个建议都应有具体依据，避免空洞评价
- **建设性** — 帮助摄影师把项目做得更好，而不是评判作品
